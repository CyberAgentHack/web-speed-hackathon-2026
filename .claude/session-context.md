## 残課題（優先順）

### 1. 投稿フロー計測不可（最大+50点）
- **原因**: 動画投稿時にFFmpeg WASMでクライアント変換 → 120秒タイムアウト超過
- **対策**: サーバーサイドで動画変換（ffmpegコマンド）に移行が最も確実
- convertMovie: 先頭5秒・正方形クロップ・10fps・無音化
- `application/client/src/utils/convert_movie.ts` → サーバー側に移行

### 2. ホームLCP=0（最大+25点）
- **原因**: React mount → APIフェッチ → 投稿カード描画の遅延でLCP候補が遅い
- **対策候補**: タイムラインAPIの先行fetch、画像のfetchpriority="high"
- スケルトンUIは入れたがLCP候補としてはまだ不十分

### 3. その他のLCP改善
- 投稿詳細LCP=1.75、写真LCP=0.25、音声LCP=2.50 → 画像の優先読み込みで改善可能
